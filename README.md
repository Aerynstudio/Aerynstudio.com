# Aeryn Studio

Site vitrine pour Aeryn Studio — vidéos FPV cinématiques (indoor & outdoor).

## Contenu
- `index.html` : page principale
- `assets/images/hero.png` : image poster
- `assets/videos/hero.mp4` : vidéo hero

## Déploiement (GitHub)
1. Initialiser le dépôt :

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <remote-url>
git push -u origin main
```

2. Pour héberger sur GitHub Pages :
- Dans les settings du repo → Pages → choisir la branche `main` et le dossier `/ (root)`.

## Remarques
- Le `poster` a été corrigé pour utiliser `assets/images/hero.png`.
- Pensez à optimiser `assets/videos/hero.mp4` pour le web si nécessaire.
